CLI Weather brings weather information to your terminal with a beautiful, minimalist interface.

## Why CLI?

Sometimes you just want to check the weather without opening a browser or an app. CLI Weather gives you that information instantly.

## Features

- Current conditions at a glance
- 7-day forecast
- ASCII art weather icons
- Support for multiple locations
- Configurable units (metric/imperial)

## Installation

```bash
cargo install cli-weather
```

## Usage

```bash
# Current weather
weather

# Specific location
weather --location "New York"

# 7-day forecast
weather --forecast
```

## Built With

- Rust for performance
- OpenWeather API for data
- Crossterm for terminal UI
